Como usar o Project Source

 

     O Project Source é um sistema onde você pode colocar seus projetos para compartilha-lo com todos os que visitam a EnergyLabs ou a Tesla Brasil! O sistema é simples de usar, porém resolvemos fazer esse tutorial explicando cada parte do Project Source.

     Quando você acessa o Project Source, você estará vendo uma página semelhante a essa:

     Nesta página você pode navegar pelas áreas as quais tem permissão livremente. Porém há funcionalidades adicionais para quem tem registro no site. Caso seja seu caso, clique em Efetuar Login e aparecerá dois campos semelhantes a estes:

     Nestes campos, digite o usuário e senha  fornecidos pelo administrador do site ou alguem autorizado.

     Após efetuar o login, você irá se deparar com novas opções a sua frente. Aqui está um exemplo do que você irá ver após efetuar seu login:

     Vamos descrever o que é cada uma das coisas novas que apareceram, começando pelas opções inferiores e depois pelas opções de pasta/arquivo.

     A primeira coisa que vemos, é o sistema informando que usuário está logado nele, e ao lado Editar Dados do Usuário onde você poderá editar o seu nome e a sua senha. Na página de edição dos dados do usuário, caso queira alterar somente o nome, deixe o campo senha em branco. Caso seja só a senha, mantenha o campo nome com o seu nome corretamente.

     A segunda coisa que podemos ver, é o Otimizar Site, que não é disponível para todos os usuários, apenas os com permissão administrativa. Está opção revisa alguns parâmetros do site, assim como miniaturas de imagens e videos.

     Abaixo, podemos ver quatro opções:

  • Criar Arquivo 
  • Enviar Arquivo
  • Criar Pasta
  • Sair

     Criar Arquivo serve para criar documentos de texto, como esse tutorial, ele conta com um editor de texto via web chamado TinyMCE.

     O editor TinyMCE é um editor HTML com quase todas as funções de processadores de texto para rodar localmente como OpenOffice e Microsoft Office. Nele, você pode criar os documentos de texto com toda liberdade e tempo que necessitar. Infelizmente por enquanto não é possível enviar imagens diretamente por ele. Por isso, é nescessário usar o Sistema de Upload do Project Source antes para enviar as imagens e assim inseri-las em seu texto. Isto será explicado com detalhes em outro tutorial.

     Visual do TinyMCE:

  

 

     Enviar Arquivo é o sistema de upload do Project Source, onde você poderá enviar Imagens, Videos, Documentos, e qualquer outro tipo de arquivo para guardar ou usar em seus documentos! Há um limite maximo do tamanho dos arquivos, e no Project Source isto está limitado a 10Mb por arquivo, porém não há limitação maximo de arquivos e nem maximo de espaço.

     Criar Pasta é o local que você usa para criar uma nova pasta. Ao clicar, aparecerá uma nova janela, pedindo um nome para colocar em sua pasta. Digite o nome e clique em criar, e sua pasta será criada.

     Obs: A pasta será criada no mesmo diretório ao qual você está vendo no momento que clica em Criar Pasta.


     Sair: Link a ser usado, após terminar seus trabalhos no site. Será efetuado o logout quando clicado nesse link, e após isso será nescessário efetuar o login novamente para poder voltar a usar seu usuário.

     Explicado as funções inferiores, explicaremos agora as funções de pasta/arquivo. Abaixo uma previsão do que você irá ver caso tenha permissões de gravação na pasta.

     Repare que três novos ícones laranjas apareceram acima do icone da pasta/arquivo. Detalharemos a função de cada um:

Ícone Excluir

     O ícone excluir serve para apagar a pasta/arquivo do servidor. Note que para se apagar uma pasta, é nescessário que ela esteja vazia.

Ícone Editar

     Este ícone serve para abrir o painel de edição de permissões/comentário do arquivo/pasta. Caso clique nele você verá um painel semelhante a esse:

     Neste painel, a opção superior permite editar o comentário do arquivo/pasta. Este comentário aparecerá quando o usuário passar o ponteiro do mouse por cima do nome ou ícone da pasta/arquivo. Caso o arquivo seja uma imagem ele irá aparecer tanto quando passar o ponteiro do mouse por cima, quanto quando for visualizada, aparecerá em baixo da imagem.

     Abaixo você pode editar as permissões do arquivo/pasta, com permissão de gravação, a pessoa terá acesso a este mesmo painel, assim como poderá editar o arquivo. Caso deseje apenas leitura para todos, marque a opção Permissão a Todos e desselecione todos os usuários no campo Permissões de Gravação.

Ícone Renomear

     Neste ícone, você poderá editar o nome da pasta ou do arquivo. Quando clicado, aparecerá um campo de texto em cima do nome do arquivo/pasta com o nome atual preenchido.

Dica: Caso seja um arquivo, aparecerá a extensão dele também, porém na hora de editar não é nescessario digitar a extensão, pois o Project Source irá automaticamente usar a extensão real do arquivo.

 

Opções Extras

 

     Opção Resumo:

     Há uma opção para colocar um pequeno texto em alguma pasta. Ele irá aparecer em baixo da listagem de arquivos e pastas:

     Como na pasta onde pode-se encontrar esse documento, você pode colocar em alguma pasta o mesmo. Acima vemos no meio da página escrito: Documentos explicativos 2

     Para se fazer isso, basta criar um arquivo chamado resumo_t na pasta. Este arquivo pode ter conteudo HTML, e pode ser editado com TinyMCE (todas as opções estarão disponíveis). Caso o usuario que entre na sua pasta não tenha acesso ao arquivo resumo_t, ele não o verá. Caso ele tenha, irá aparecer assim:


     Facil não? Outra coisa que você pode fazer, é adicionar uma imagem ao rodapé, para aparecer para a pasta onde ela está. Funciona do mesmo jeito que o resumo_t porém é um arquivo logo_t.jpg. Nescessariamente o arquivo precisa ser um JPEG, caso não for, não irá funcionar.
